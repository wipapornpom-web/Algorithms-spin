# Algorithm Spin Challenge - Motion Only Camera Edition

เกมการศึกษา HTML5 สำหรับนักเรียนระดับประถมศึกษา ป.1–ป.3 เพื่อฝึกแนวคิด **อัลกอริทึม (Algorithm)** ผ่านการคิดเป็นลำดับขั้นตอน

## จุดเด่นเวอร์ชันนี้

- เล่นแบบ **ไม่ใช้เมาส์** และ **ไม่สัมผัสหน้าจอ**
- ใช้กล้องเว็บแคมส่องคนเล่นบนหน้าจอ
- คงระบบ **วงล้อสุ่มภารกิจ** เหมือนเดิม
- ควบคุมด้วยการเคลื่อนไหวของร่างกาย
- ใช้ HTML, CSS, JavaScript แบบ Vanilla
- ไม่ใช้ Framework
- อัปโหลดขึ้น GitHub Pages ได้ทันที

## วิธีควบคุม

| ท่าทาง | คำสั่งในเกม |
|---|---|
| โบกมือซ้าย-ขวา | เริ่มเกม / หมุนวงล้อ |
| ขยับตัวหรือมือไปซ้าย | เลื่อนตัวเลือกไปทางซ้าย / ก่อนหน้า |
| ขยับตัวหรือมือไปขวา | เลื่อนตัวเลือกไปทางขวา / ถัดไป |
| ยกมือสูง | เลือกคำตอบที่มีกรอบสีเหลือง |
| ชูสองมือ หรือ กระโดด | ตรวจคำตอบ |
| โบกมือหรือขยับแรงในหน้าวิธีเล่น/คะแนน | กลับหน้าแรก |

## วิธีใช้งาน

1. แตกไฟล์ ZIP
2. เปิด `index.html` ด้วย Chrome หรือ Microsoft Edge
3. กดอนุญาตการใช้กล้องเมื่อเบราว์เซอร์ถาม
4. ยืนห่างกล้องประมาณ 1.5–3 เมตร
5. ให้กล้องเห็นช่วงศีรษะ ไหล่ แขน และลำตัว
6. ใช้ท่าทางตามตารางเพื่อเล่นเกม

> หมายเหตุ: หากเปิดผ่าน GitHub Pages ต้องใช้ HTTPS ซึ่ง GitHub Pages รองรับอยู่แล้ว กล้องจึงทำงานได้ตามปกติ

## อัปโหลดขึ้น GitHub Pages

1. สร้าง Repository ใหม่ใน GitHub
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้
3. ไปที่ Settings > Pages
4. เลือก Branch: `main`
5. เลือก Folder: `/root`
6. กด Save
7. รอสักครู่แล้วเปิดลิงก์ GitHub Pages
