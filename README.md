# Algorithm Spin Challenge - Full Camera Pointer

เวอร์ชันนี้ปรับตามแนวทางใช้ง่ายสำหรับเด็ก ป.1–ป.3

## จุดเด่น

- กล้องเป็นพื้นหลังเต็มจอ เห็นคนเล่นเต็ม ๆ
- หน้าแรกเป็นวงล้อทันที
- โบกมือเพื่อหมุนวงล้อ
- วงล้อหยุดที่ภารกิจไหน เข้าเล่นภารกิจนั้นทันที
- ในเกมใช้มือชี้/ขยับไปที่กล่องคำสั่ง แล้วค้างไว้ประมาณ 1 วินาที
- ไม่ต้องใช้เมาส์
- ไม่ใช้ Framework
- ไม่ใช้ไลบรารีภายนอก
- อัปขึ้น GitHub Pages ได้

## วิธีเล่น

1. เปิดเว็บและอนุญาตกล้อง
2. ยืนห่างจากกล้องประมาณ 1.5–2.5 เมตร
3. โบกมือเพื่อหมุนวงล้อ
4. เมื่อเข้าเกม ให้ใช้มือชี้ไปที่กล่องคำสั่งด้านล่าง
   - เลี้ยวซ้าย
   - เดิน
   - เลี้ยวขวา
   - กลับวงล้อ
5. ชี้ค้างไว้ประมาณ 1 วินาที เกมจะสั่งหุ่นยนต์ให้ทำตาม

## ภารกิจ

- เก็บเพชร
- ไปถึงธง
- ส่งของ
- เก็บผลไม้
- เก็บดาว
