# Algorithm Spin Challenge - Finger Drag Stable Edition

เวอร์ชันนี้ปรับให้เล่นง่ายและเสถียรกว่าเดิม

## จุดเด่น

- กล้องเต็มจอเป็นพื้นหลัง
- วงล้ออยู่หน้าแรกทันที
- วงล้อมีชื่อเกมบนแต่ละช่อง
- โบกมือเพื่อหมุนวงล้อ
- สุ่มเกมแบบไม่ซ้ำจนกว่าจะครบทุกเกม
- ทุกเกมใช้รูปแบบเดียว: ใช้นิ้วชี้ลากตามเส้นทาง
- หากลากถึงปลายทาง/ธง จะผ่านด่านและกลับไปวงล้ออัตโนมัติ
- ไม่ต้องใช้เมาส์ในการเล่นจริง

## เกมทั้งหมด 10 เกม

1. เก็บเพชร
2. ไปหาธง
3. ส่งของ
4. ปลูกต้นไม้
5. แปรงฟัน
6. พิซซ่า
7. จรวด
8. เปิดคอม
9. เก็บของเล่น
10. ล้างมือ

## วิธีเล่น

1. เปิดเว็บผ่าน GitHub Pages หรือ HTTPS
2. อนุญาตให้เว็บใช้กล้อง
3. โบกมือเพื่อหมุนวงล้อ
4. วงล้อหยุดที่เกมไหน จะเข้าเกมนั้นทันที
5. ใช้นิ้วชี้ลากตามทางสีม่วง
6. ลากถึงธง/ปลายทาง = ผ่านด่าน
7. เกมกลับไปที่วงล้อ เพื่อหมุนรอบต่อไป

## อัปขึ้น GitHub Pages

1. แตกไฟล์ ZIP
2. อัปโหลดไฟล์ทั้งหมดเข้า Repository
3. ไปที่ Settings > Pages
4. เลือก Branch เป็น main และ Folder เป็น /root
5. กด Save
6. เปิดลิงก์ GitHub Pages แล้วอนุญาตกล้อง

## หมายเหตุ

ระบบใช้ MediaPipe Hands จาก CDN ของ Google จึงต้องใช้อินเทอร์เน็ตตอนเปิดเกมครั้งแรก และต้องเปิดผ่าน HTTPS เพื่อให้กล้องทำงานได้ถูกต้อง
