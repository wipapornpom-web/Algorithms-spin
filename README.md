# Algorithm Spin Challenge - Simple Motion

เกมการศึกษา HTML5 สำหรับ ป.1–ป.3 แบบใช้กล้อง ไม่เน้นตัวหนังสือ

## วิธีเล่น

1. เปิด `index.html` หรืออัปโหลดขึ้น GitHub Pages
2. อนุญาตให้เว็บใช้กล้อง
3. หน้าแรกจะเห็นวงล้อและกล้อง
4. โบกมือแรง ๆ เพื่อหมุนวงล้อ
5. วงล้อสุ่มภารกิจ แล้วเข้าเกมอัตโนมัติ

## ท่าทาง

- 👋 โบกมือแรง ๆ = หมุนวงล้อ
- 🙋 ยกมือซ้าย = หุ่นยนต์เลี้ยวซ้าย
- 🙋 ยกมือขวา = หุ่นยนต์เลี้ยวขวา
- 🚶 เดินอยู่กับที่ / ขยับขา = หุ่นยนต์เดินหน้า
- 🙌 ชูสองมือ = กลับวงล้อ

## หมายเหตุสำคัญ

- กล้องจะทำงานดีที่สุดเมื่อเปิดผ่าน HTTPS เช่น GitHub Pages
- ถ้าเปิดจากไฟล์ในเครื่อง บางเบราว์เซอร์อาจไม่อนุญาตให้ใช้กล้อง
- โค้ดนี้ไม่ใช้ MediaPipe และไม่ใช้ Framework เพื่อให้เปิดง่าย
- มีคีย์บอร์ดสำรองสำหรับครูทดสอบ: Space หมุน, A ซ้าย, D ขวา, W เดิน, B กลับวงล้อ
