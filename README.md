# Algorithm Spin Challenge - Motion Only Camera Edition

เกมการศึกษา HTML5 สำหรับนักเรียน ป.1–ป.3 เรื่อง “อัลกอริทึม” เล่นผ่านเว็บเบราว์เซอร์โดยไม่ใช้เมาส์และไม่สัมผัสจอ

## จุดเด่น

- วงล้อสุ่มภารกิจเหมือนเดิม
- มีกล้องส่องคนเล่นบนหน้าจอ
- เล่นด้วยการเคลื่อนไหวหน้ากล้อง
- 5 ภารกิจหลักเพื่อให้ใช้งานจริงง่ายขึ้น
- ใช้ HTML, CSS, JavaScript แบบ Vanilla
- อัปโหลดขึ้น GitHub Pages ได้ทันที

## วิธีเล่น

1. เปิด `index.html` ผ่านเว็บเบราว์เซอร์ Chrome หรือ Microsoft Edge
2. กดอนุญาตให้ใช้กล้อง
3. ยืนหน้ากล้องประมาณ 1.5–3 เมตร
4. ใช้ท่าทางเล่น

| ท่าทาง | คำสั่ง |
|---|---|
| โบกมือ/ขยับกลางจอ | เริ่มเกม / หมุนวงล้อ / ยืนยันคำตอบ |
| ขยับฝั่งซ้ายของจอ | เลือกคำตอบก่อนหน้า |
| ขยับฝั่งขวาของจอ | เลือกคำตอบถัดไป |

## ภารกิจ 5 ด่าน

1. Robot Maze
2. Debug
3. Order
4. Rocket
5. Treasure

## วิธีอัปโหลด GitHub Pages

1. สร้าง Repository ใหม่ใน GitHub
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้
3. ไปที่ Settings > Pages
4. เลือก Branch เป็น `main` และ Folder เป็น `/root`
5. กด Save
6. เปิดลิงก์ GitHub Pages ที่ระบบสร้างให้

## หมายเหตุสำคัญ

ระบบกล้องต้องเปิดผ่าน `https://` หรือ `localhost` จึงจะทำงานได้ดี หากเปิดจากไฟล์โดยตรงแล้วกล้องไม่ขึ้น ให้ใช้งานผ่าน GitHub Pages
