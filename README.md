# Algorithm Spin Challenge - Drag Pointer Edition

เวอร์ชันนี้ปรับให้เล่นด้วย **มือชี้ + จีบนิ้วเพื่อจับ/ลาก/ปล่อย** เป็นหลัก และเพิ่ม **ชื่อเกมบนวงล้อ** กับ **เส้นทางถนนในเกมที่ต้องเดินตามทาง**

## วิธีเล่น

1. เปิดไฟล์ `index.html` ผ่าน GitHub Pages หรือ HTTPS
2. อนุญาตให้เว็บใช้กล้อง
3. กล้องจะแสดงเต็มจอเป็นพื้นหลัง
4. ใช้นิ้วชี้ไปที่ปุ่มกลางวงล้อ แล้วจีบนิ้วโป้งกับนิ้วชี้เพื่อหมุนวงล้อ
5. เมื่อเข้าเกม ให้จีบนิ้วเพื่อจับรูปภาพ แล้วลากไปวางในช่อง/เป้าหมาย
6. เกมที่เป็นเส้นทาง เช่น เก็บเพชร/ไปธง/ไปโรงเรียน ให้ลากตัวละครตามถนนและเก็บเป้าหมายตามหมายเลข
7. จีบที่ปุ่ม “ตรวจคำตอบ” เพื่อตรวจคำตอบ
8. ผ่านด่านแล้วกลับไปหมุนวงล้อใหม่

## เกมในเวอร์ชันนี้

มี 10 ภารกิจแบบลาก/เคลื่อนย้ายวัตถุทั้งหมด

- แปรงฟัน
- ทำพิซซ่า
- ปล่อยจรวด
- ปลูกต้นไม้
- เปิดคอมพิวเตอร์
- ล้างมือ
- เก็บของเล่น
- เก็บเพชร
- ไปหาธง
- ไปโรงเรียน

## หมายเหตุสำคัญ

- กล้องทำงานได้ดีที่สุดเมื่อเปิดผ่าน `https://` เช่น GitHub Pages
- ถ้าเปิดจากไฟล์ในเครื่องโดยตรง บางเบราว์เซอร์อาจไม่อนุญาตให้ใช้กล้อง
- ใช้ Chrome หรือ Microsoft Edge
- ควรมีแสงสว่างพอ และให้มืออยู่ในกล้องชัดเจน
