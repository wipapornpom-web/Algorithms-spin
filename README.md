# Algorithm Spin Challenge - Mixed Motion

เกมการศึกษา HTML5 สำหรับ ป.1–ป.3 ใช้กล้องเต็มจอ + วงล้อ + ท่าทางสลับกันในแต่ละเกม

## วิธีเล่น
1. เปิด `index.html` ผ่าน GitHub Pages หรือ HTTPS
2. อนุญาตให้ใช้กล้อง
3. โบกมือเพื่อหมุนวงล้อ
4. วงล้อสุ่มเข้าเกมอัตโนมัติ

## ท่าทางในเกม
- 👋 โบกมือ = หมุนวงล้อ
- 👆 นิ้วชี้ลาก = เกมเรียงภาพ / ลากของ
- ↔️ เอียงตัว = เลี้ยวซ้าย/ขวา หรือรับผลไม้
- ⬇️ ก้มตัว = หุ่นยนต์เดินหน้า
- 🦘 กระโดดเบา ๆ = เก็บดาว/เก็บเพชร
- 🙌 ชูสองมือ = ปล่อยจรวด

## หมายเหตุสำคัญ
- ต้องใช้อินเทอร์เน็ต เพราะเรียก MediaPipe ผ่าน CDN
- กล้องมักไม่ทำงานถ้าเปิดเป็น `file://` ให้ใช้งานบน GitHub Pages หรือ localhost
- Chrome / Edge เหมาะที่สุด

## อัปโหลด GitHub Pages
1. สร้าง Repository ใหม่
2. อัปโหลดไฟล์ทั้งหมด
3. Settings > Pages > Deploy from branch > main / root
4. เปิดลิงก์ Pages แล้วอนุญาตกล้อง
